# Refact v8.2.1

Refact v8.2.1 is a daemon-focused patch on top of v8.2.0: it stops a project-worker restart loop, hardens the daemon's LAN authentication, and improves the Refact Daemon page with live logs and review fixes.

> [!WARNING]
> **Breaking change — local daemon**
> The plugins require and launch a local `refact daemon` (`127.0.0.1:8488`) that supervises per-project workers. The engine binary is downloaded on first run from GitHub releases and is no longer bundled in the plugin; the in-IDE webview connects over loopback. If you used a pre-8.2.0 bundled-engine version, first start now downloads the engine.

> [!IMPORTANT]
> **Early preview**
> The daemon and TUI are early preview and may be buggy. IDE plugins and the engine are the stable surfaces.

## Highlights

- Daemon worker stability: fixed a project-worker restart loop so idle/restarting workers settle instead of cycling.
- Daemon LAN auth hardening: tightened authentication for non-loopback (LAN) daemon binds, with explicit LAN auth settings.
- Refact Daemon page: added live daemon/worker logs plus review fixes for the daemon status view.

## Install

Unix/macOS:

```sh
curl -fsSL https://raw.githubusercontent.com/JegernOUTT/refact/main/install.sh | sh
```

Windows PowerShell:

```powershell
irm https://raw.githubusercontent.com/JegernOUTT/refact/main/install.ps1 | iex
```

Start Refact with:

```sh
refact
refact tui
```
