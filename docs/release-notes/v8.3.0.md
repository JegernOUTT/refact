# Refact v8.3.0

Refact v8.3.0 improves model support and responsiveness across chat, CodeGraph, and worktree workflows.

## Highlights

- OpenAI Codex providers now recognize GPT-5.6 Sol, Terra, and Luna, including their model-specific context windows, output limits, tool support, and available reasoning-effort levels.
- Codex reasoning streaming now requests sequential-cutoff summaries and treats completed summary events as authoritative, replacing partial headings with the full reasoning summary in the engine, GUI, and TUI.
- Large chat snapshots can load without the previous 8 MiB parser ceiling, while parameter synchronization avoids duplicate requests and no longer blocks sends.
- CodeGraph analysis tools are available in read-only modes and accept the same workspace, absolute, and active-worktree path forms as file-reading tools.
- Worktree and trajectory filesystem processing moves off asynchronous request paths, reducing stalls during chat, agent, merge, and history operations.
