# v8.4.0

v8.4.0 ships a standalone web dashboard for managing Refact without an IDE, installable via Homebrew/winget/Scoop/npm/Docker, alongside a major MCP overhaul, faster code intelligence, and broad UI polish.

## Highlights

- **Standalone Web Dashboard** — Run `refact ui` to open a full management dashboard in your browser. Manage multiple projects, browse files, use an integrated terminal, view git status and diffs, monitor scheduled jobs, track usage, and run the AI agent — all without an IDE.
- **Install everywhere** — Refact is now distributed through Homebrew, winget, Scoop, npm, and Docker, so you can install it with one command on any platform.
- **MCP Marketplace & transport overhaul** — The MCP server experience is rebuilt: full HTTP/SSE/stdio transport parity, a real marketplace with install/update/uninstall flows, OAuth auto-discovery, credential prompts, and connection testing.
- **10–500× faster code intelligence** — CodeGraph overview, dead-code detection, duplication analysis, health checks, and git-risk signals now run non-blocking in the background, dropping response times from minutes to under a second.
- **Tool toggle UI for modes** — Customizing agent modes now shows every available tool as a searchable on/off switch with descriptions, replacing the old free-text list.

## Web Dashboard & CLI

- New `refact ui` command opens the dashboard in your default browser (with `--json`/`--no-open` flags for scripting).
- Dashboard home page with first-run wizard for new users.
- Projects grid: add, browse, and drill into project detail pages with introspection tabs.
- Integrated file explorer with syntax-highlighted read-only viewer.
- Full interactive terminal panel (xterm.js) with tabs and session reattachment.
- Git panel: file status, stage/unstage, diffs, commit, branch history, and worktree management.
- Cross-project scheduler, usage, and activity pages.
- Doctor page: automated health checks with one-click fixes for common issues.
- Settings page with network/access, self-update, and remote daemon restart.

## MCP

- Full transport parity across HTTP, SSE, and stdio backends.
- Marketplace rebuilt with real install/update/uninstall flows and setup wizard.
- OAuth auto-discovery: auth fields are now advanced/hidden when the server supports automatic login.
- OAuth state management stabilized in the engine.
- Import MCP servers from files and existing projects.
- Hardened marketplace, import/export, and session security.

## Agent & Planner

- Tool toggle list for mode customization with search and descriptions (#22).
- Planner self-heals when notification metadata is lost and wakes immediately on blocking sub-agent questions.
- Silently-dropped tools in setup modes and planner discovery are now surfaced correctly.
- Subagent, delegate, and buddy child chats are properly restored when opening their trajectories.
- Model-scoped weekly usage windows for Claude Code provider.

## Performance

- Code intelligence tools (graph overview, dead code, duplication, health, git risk) run non-blocking and ~10–500× faster.
